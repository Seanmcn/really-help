ReallyHelp.org uses publicly accessible data to try and make charities more transparent.

Our way of processing the data isn't perfect and should be taken with a grain of salt, you can read about assumptions we make here.

ReallyHelp.org is an Open Source and non-commercial product. If you'd like to help us out please check our Github or see our donation page.
